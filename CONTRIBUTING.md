# Contributing to Skipper Logbook

Thanks for your interest in improving Skipper Logbook. This is a native iOS app
(SwiftUI + SwiftData) built with an [XcodeGen](https://github.com/yonyz/XcodeGen)
project spec.

## Getting set up

You need **macOS + Xcode 15+** (iOS 17 SDK) and XcodeGen:

```bash
brew install xcodegen
xcodegen generate
open SkipperLogbook.xcodeproj
```

See [`SETUP.md`](SETUP.md) for signing, the App Group / Live Activity setup, and the
manual test checklist.

## Project layout

- `SkipperLogbook/App/` — app entry, router, root shell
- `SkipperLogbook/Core/` — Location, NavigationMath, Persistence, Theme, Widget,
  Permissions, Export
- `SkipperLogbook/Shared/` — SwiftData models, design system, reusable components,
  extensions, the widget snapshot/attributes (shared with the extension)
- `SkipperLogbook/Features/` — one folder per screen
- `SkipperWidgets/` — the WidgetKit + Live Activity extension
- `SkipperLogbookTests/` — unit tests

## Ground rules

- **Never commit `SkipperLogbook.xcodeproj`** — it is generated by XcodeGen. Change
  `project.yml` instead and re-run `xcodegen generate`.
- **Design system first.** Don't hardcode colors, fonts, radii or spacing — use
  `@Environment(\.appTheme)`, `AppFont`, `Spacing`. One strong accent per screen; MOB
  red is reserved for emergencies only.
- **Localize everything.** Every user-facing string is a key in
  `Resources/Localizable.xcstrings` with both `en` and `ru`.
- **Keep it honest.** Unimplemented features are marked `Coming soon` via
  `.comingSoon()` — never faked.
- **`@Observable` + Observation**, not `ObservableObject`. `@MainActor` on view models
  and the location/voyage engines.
- **Add tests** for pure logic (navigation math, recorders, seeding).

## Commits & PRs

- Use clear, present-tense commit messages (Conventional Commits welcome:
  `feat:`, `fix:`, `docs:`, `refactor:`, `test:`).
- Keep PRs focused. Describe the change, how you tested it, and include a screenshot
  for UI changes.
- CI runs `xcodegen generate`, builds, and runs the test suite on a macOS runner —
  make sure it's green.

## Safety note

Skipper Logbook is an assistive logbook, **not** certified navigation. Contributions
must not present it as a replacement for charts/ECDIS or imply safety-of-navigation
guarantees.
