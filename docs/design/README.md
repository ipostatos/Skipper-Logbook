# Design references

These are the source mockups and explorations that shaped Skipper Logbook's UI.
They document the intended look & feel and the evolution of the visual direction —
from early concepts to the shipped **"Liquid Nautical"** language (light, Apple-native,
one strong accent per screen, MOB-red reserved for emergencies).

They are references only; the app's actual styling lives in the design system under
[`SkipperLogbook/Shared/DesignSystem`](../../SkipperLogbook/Shared/DesignSystem) and its
reusable components.
